# 📊 Student Performance Analysis

## 📌 Project Overview
This project analyzes student exam performance using Python, Pandas, and Matplotlib. The goal is to understand how different factors influence students' exam scores.

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Google Colab

## 📂 Dataset
- exams.csv
- 1000 student records

## 📈 Analysis Performed
- Data Exploration
- Missing Value Analysis
- Subject-wise Average Scores
- Gender-wise Performance
- Test Preparation Analysis
- Lunch Type Analysis
- Parental Education Analysis
- Correlation Analysis

## 🔍 Key Findings
- Reading and Writing scores were generally higher than Mathematics scores.
- Students who completed the test preparation course scored higher.
- Students with standard lunch had better average scores.
- Reading and Writing scores showed a strong positive correlation.

## 🎯 Conclusion
This project demonstrates the use of exploratory data analysis (EDA) to discover meaningful insights from educational data.
